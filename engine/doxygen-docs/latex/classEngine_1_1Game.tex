\subsection{Detailed Description}
\doxyref{Game}{p.}{classEngine_1_1Game} base class. Any game using this engine makes an instance of an inherited \doxyref{Game}{p.}{classEngine_1_1Game} class. This class handles the initialization of all subsystems like video, audio and so on, and it stores all the loaded resources in the resource managers. The member function \doxyref{onInit()}{p.}{classEngine_1_1Game_f6097e589feccb0a9922c005a6ab4524} needs to be inherited and implemented by the users of the engine.
